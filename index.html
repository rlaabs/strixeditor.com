FAQ

What file formats does Strix SQLite open?

Strix opens any SQLite database file — including .sqlite, .sqlite3, .db, and custom extensions. You can also import CSV, TSV, JSON, and Excel (.xlsx) files; multi-sheet workbooks expand into one SQLite table per sheet.

Is Strix SQLite free?

No — Strix is a paid app with a one-time purchase of $9.99 for Mac and $9.99 for iPad. There are no subscriptions and no in-app purchases. See the pricing page for details.

Is Strix SQLite safe to use on production databases?

Yes. Databases open in read-only mode by default, so you cannot accidentally modify data. Switching to edit mode highlights every change until you explicitly save or discard it. Strix does not transmit any data — all databases stay on your device.

Does Strix SQLite require macOS or iPadOS?

Yes. Strix is built natively with Swift and SwiftUI and runs on macOS 14 (Sonoma) or later, and iPadOS 18 or later. There is no Windows or Linux version.

Does it support full-text search?

Yes. Strix uses SQLite's FTS5 engine for fast full-text search across large databases, with result highlighting and real-time filtering.

Can I buy Strix outside the App Store?

Yes — Strix for Mac is available directly via Paddle for $9.99, with the same features as the App Store version.
